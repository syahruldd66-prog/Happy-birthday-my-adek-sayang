MUSIC FOLDER
============
Place your MP3 files here:

  music/song1.mp3
  music/song2.mp3
  music/song3.mp3

Configure start/end times and volume in:
  js/audio-system.js → MUSIC_LIBRARY array

Example custom timing:
  startTime: 45,    // start at 0:45
  endTime:   130,   // end at 2:10
  loopStart: 45,    // loop back to start point
  volume:    0.55,
  loop:      true
